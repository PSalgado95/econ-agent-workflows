# <PROJECT_NAME>

This README is a minimal project index, not a live status memo.

## Folder Map

- `code/` - analysis code, scripts, notebooks, and project helpers.
- `data/raw/` - raw inputs. Treat this folder as read-only.
- `data/processed/` - derived datasets and cleaned intermediate files.
- `output/results/` - result tables, checks, and machine-readable outputs.
- `output/tables/` - reader-facing tables.
- `output/figures/` - figures and plots.
- `output/logs/` - run logs.
- `paper/` - paper draft material when the project reaches that stage.
- `reference/` - papers, manuals, codebooks, and other reference material.
- `scratch/docs/` - non-authoritative working plans, brainstorms, memos, and notes.
- `scratch/agents/` - temporary agent clutter such as screenshots, package stages, return extracts, and raw logs.

## Working Rules

- Do not modify raw data in place.
- Keep derived data out of `data/raw/`.
- Keep temporary agent material in `scratch/`.
- Promote a scratch note into a tracked file only when explicitly requested.
